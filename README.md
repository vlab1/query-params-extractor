# Query Params Extractor

**Deploy to main:**

```bash
git add . && git commit -m "deploy" && git push origin main
```

**Deploy to gh-pages:**

```bash
cd .. ; npm run build && cd build && git init && git remote add origin https://github.com/vlab1/query-params-extractor.git && git add . && git commit -m "deploy" && git push -f origin HEAD:gh-pages && cd ..
```
